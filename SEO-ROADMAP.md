## フェーズ別ロードマップ

### 📅 Phase 1: 基盤整備（2026/5/18 - 6/15、約4週間）

**ゴール: クロール・インデックス完了、E-E-A-T基礎、ローカルSEO着手**

#### 技術SEO
- [ ] 構造化データ（JSON-LD）追加
  - `LocalBusiness` schema（運営者情報・営業時間・サービスエリア）
  - `Service` schema（各SEO LPに）
  - `FAQPage` schema（各LPのFAQセクションに）
  - `BreadcrumbList` schema
- [ ] Core Web Vitals チェック（PageSpeed Insights）
  - LCP < 2.5s, CLS < 0.1, INP < 200ms
- [ ] 画像最適化（WebP化、適切な alt 属性、サイズ最適化）
- [ ] 内部リンク構造の強化（pillar-cluster モデル）
  - index.html を pillar、各 SEO LP を cluster としてリンク
  - 各 LP 同士も関連ページとして相互リンク

#### E-E-A-T 強化
- [ ] **運営者プロフィールページ作成**（バリスタの経歴・受賞歴・資格）
- [ ] **実績ページ作成**（過去のイベント実績一覧）
- [ ] お客様の声（testimonials）セクションを各LPに追加
  - 最低3〜5件、可能なら写真付き
- [ ] 「会社概要」「特定商取引法」は既存活用

#### ローカルSEO（重要）
- [ ] **Googleビジネスプロフィール（GBP）登録**
  - サービスエリア型ビジネスとして登録（実店舗無しの場合）
  - サービスエリア: 東京都・神奈川県・千葉県・埼玉県
  - カテゴリ: 「ケータリングサービス」「カフェ」
  - 営業時間、電話番号、サービス内容、写真10枚以上
- [ ] **NAP統一**（Name/Address/Phone を全媒体で揃える）
  - サイト、GBP、SNS、各種ディレクトリで同一表記
- [ ] 「コーヒー ケータリング 東京」等で見える各種ディレクトリへの登録
  - Googleマップ
  - Yahoo!プレイス
  - エキテン
  - 開業マッチ／みんなのウェディング等の業種別

### 📅 Phase 2: コンテンツ拡充（2026/6/16 - 7/31、約6週間）

**ゴール: コンテンツ量で権威性確立、ロングテール獲得**

#### 事例コンテンツ（最重要・最強の被リンク資産）
- [ ] **イベントレポート記事を週1本ペース（計6〜8本）**
  - 各記事1500〜2500字、写真5枚以上
  - URL構造: `/case/{slug}.html` 推奨
  - 各記事から該当する SEO LP へ内部リンク
  - 想定タイトル例:
    - 「【事例】〇〇展示会で200名にコーヒー提供したケース」
    - 「【事例】結婚式の披露宴でラテアートサービスを実施」
    - 「【事例】〇〇社の社内懇親会でバリスタ派遣」

#### FAQ拡充
- [ ] 各SEO LPに **FAQセクション10問以上** 追加
  - 想定質問: 料金体系、最低人数、機材、停電・水道なしOKか、何ドリンク提供できるか、雨天時、当日キャンセル等
- [ ] **「よくある質問」総合ページ作成**（`faq.html`）→ 各LPからリンク

#### ブログ記事（KW狙い）
- [ ] サブカテゴリ別の解説記事を **月3〜4本**
  - 「展示会でコーヒーを提供するメリット5選」
  - 「結婚式でコーヒーケータリングを選ぶときの3つのポイント」
  - 「社内イベントのバリスタ派遣の費用相場」
  - 「ラテアート出張の演出アイデア集」
  - 「コーヒー出張の機材・電源・水道事情」

#### 動画・ビジュアル
- [ ] YouTubeチャンネル開設（実演動画3本以上）
- [ ] Instagram投稿（最低週2投稿）
- [ ] 各LP内にYouTube動画を埋め込み（滞在時間UP・E-E-A-T強化）

### 📅 Phase 4: 最適化と総仕上げ（2026/9/1 - 9/30、約4週間）

**ゴール: 細かな最適化で順位を押し上げ、目標達成**

#### データドリブン最適化
- [ ] GSC「検索パフォーマンス」を分析
  - 表示回数あり / クリック少ない → titleとmeta descriptionを書き換え（CTR最適化）
  - 平均掲載順位 4〜10位のキーワード → ピンポイントでコンテンツ強化（あと一押しで上位）
- [ ] GA4「ランディングページ」分析
  - 直帰率高いLP → ファーストビュー・CTA・スクロール深度を改善

#### コンテンツ強化（個別)
- [ ] 各 SEO LP の文字数を **2500〜4000字** まで拡充
  - 比較表、料金シミュレーター、地域別ガイド等を追加
- [ ] LP内に「関連実績」セクションを追加（Phase 2のイベントレポートと連動）
- [ ] 構造化データの追加検証（リッチリザルトテスト全ページ通過確認）

#### テクニカル仕上げ
- [ ] 不要なリダイレクト・404 の確認
- [ ] CWV再測定 → 改善
- [ ] hreflang 対応（英語版作る場合）

#### 想定外への備え

## リスクとシナリオ

### 達成見込み（高）
- Tier A（ロングテール）: 7キーワード中5〜6個で **3位以内達成** が現実的

### 達成見込み（中）
- Tier B: top10 入りは現実的、top3 はコンテンツ量と被リンク次第
- ローカル「東京 バリスタ派遣」「東京 コーヒーケータリング」はGBP施策が効けば top5 圏内

### 達成見込み（低 / 9月時点では難しい）
- Tier C（ヘッド）: top3 は通常1〜2年。9月時点は top10〜20 が現実的
  - 9月以降、継続施策で来年春までに top3 を狙う

### 致命的になり得るリスク
- Googleコアアップデート: 月単位で順位が大変動。Phase 3〜4の時期に重なると影響大
- 競合の参入: 同業他社のSEO強化、新規参入
- E-E-A-T の証拠不足: 運営者情報・実績の透明性が低いと評価上がりにくい
